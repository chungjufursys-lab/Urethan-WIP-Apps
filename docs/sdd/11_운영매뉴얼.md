# 11. 운영매뉴얼

## 현재 운영 절차
1. 앱 실행
2. 관리자 로그인
3. `가공실적등록.xls` 업로드 또는 기본 파일 재불러오기
4. 조회 화면에서 부족/위험도 확인

## 운영 시 주의사항
- 현재 구조에서는 제품코드와 부품코드 관계를 전용 구조로 점검할 수 없다.
- 현재 구조에서는 부품명 표기 흔들림으로 동일 부품이 분산 인식될 수 있다.

## 구조 재정의 후 운영 기준 초안
- 조회는 항상 `정규화 부품명 + 색상`으로 수행한다.
- 제품코드는 보조 정보 및 캘린더 표시용으로 사용한다.
- 제품코드와 부품코드 관계는 매핑 테이블에서 점검한다.
- 업로드 후 부품명 정규화 결과를 우선 점검한다.
- 날짜 해석은 `최초투입일자=위험도`, `포장일자=화면노출`로 고정한다.

- 운영 중 데이터 정정 절차
- 기존 데이터 백업/복구 절차
