# 01. 프로젝트 개요

## 목적

- 우레탄 품목 재공을 더 컴팩트하게 관리할 수 있는 운영용 MVP를 제공한다.
- 공개 조회 화면과 관리자 수정 화면을 분리해 외주업체와 내부 담당자가 같은 기준 데이터를 보되 수정 권한은 제한한다.
- 생산계획과 현재 재공을 연결해 부족분, 납기 위험, 외주 공유 정보를 한 화면에서 확인할 수 있게 한다.

## 현재 확인된 사실

- 기술 스택은 Python + Streamlit + SQLite다.
- 관리자는 정식 회원가입 없이 비밀번호로만 인증한다.
- 외주업체 조회 화면은 로그인 없이 공개한다.
- 생산계획 샘플 원본은 `공정진행정보.xls`다.
- 실제 엑셀에서는 `생산라인`에 `우레탄`이 포함되고 `건명`에 `외주입고분`이 포함된 행이 현재 사용자 설명의 필터 조건과 가장 가깝다.

## MVP 범위

- 조회, 입력, 수정, 삭제, 계획 업로드, 부족분 계산, CSV 다운로드
- 샘플 데이터 자동 적재
- 변경시각/수정자 저장

## MVP 제외 범위

- 정식 계정 시스템
- 모바일 최적화 전용 UI
- ERP 직접 연동
- 대용량 권한 체계
