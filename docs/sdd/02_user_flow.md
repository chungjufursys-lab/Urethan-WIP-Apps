# 02. 화면 구조 및 사용자 흐름

## 메뉴 구조

1. 대시보드
2. 재공 현황 조회
3. 생산계획 / 부족분 조회
4. 외주 공유 화면
5. 관리자 로그인
6. 관리자 전용: 재공 입력관리
7. 관리자 전용: 품목 마스터
8. 관리자 전용: 위치 마스터
9. 관리자 전용: 생산계획 관리

## 사용자 흐름

### 공개 사용자

1. 앱 접속
2. 대시보드에서 현재 재공, 부족 품목 수, 위험 일정 확인
3. 재공 현황 조회에서 품목/색상/위치/상태 기준 필터 조회
4. 생산계획 / 부족분 조회에서 부족 현황과 납기 위험도 확인
5. 외주 공유 화면에서 업체 기준 요청량, 납기, 상태를 확인하고 CSV로 다운로드

### 관리자

1. 관리자 로그인 메뉴 진입
2. 비밀번호와 수정자명을 입력해 세션 인증
3. 재공 입력관리에서 재공 데이터 추가/수정/삭제
4. 품목 마스터에서 품목 및 색상 변형 관리
5. 위치 마스터에서 보관 위치 기준정보 관리
6. 생산계획 관리에서 수기 입력 또는 엑셀 업로드
7. 변경 내용은 수정자와 변경시각을 함께 저장

## UX 원칙

- 숫자/표 중심의 운영보드
- 부족/위험/긴급 상태는 색상 배지로 강조
- 필터가 우선이고 입력 폼은 단순하게 유지
